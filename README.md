# My-Quant-Project

这是我尝试开发的量化交易系统，用于个人练习 python 和量化交易相关内容。

## 项目结构

- `data/` 用于存放的 CSV 或数据库文件
- `strategies/` 存放不同的策略脚本
- `engine/` 核心回测逻辑或执行逻辑
- `utils/` 工具类（绘图、日期处理、指标计算）
- `storage/` 专门存放数据的仓库
    - `raw/` 原始行情数据
    - `processed/` 处理后的数据
- `scripts/` 存放逻辑脚本
- `notebooks/` 存放 .ipynb 文件进行实验
- `main.py` 项目入口
- `requirements.txt` 依赖包列表
